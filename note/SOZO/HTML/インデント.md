## インデント
* コードを字下げするという意味
* 可読性を高めるために階層を作る
```
<div>
<h1>私のポートフォリオ</h1>
<ul>
<li>ABOUT</li>
<li>PORTFOLIO</li>
<li>CONTACT</li>
</ul>
</div>
```
```html
<div>
 <h1>私のポートフォリオ</h1>
 <ul>
  <li>ABOUT</li>
  <li>PORTFOLIO</li>
  <li>CONTACT</li>
 </ul>
</div>
```
### 入力方法
* tab:一回の字下げでいい場合があったり入力が楽。環境によって左右される
* 半角スペース:環境によって左右される幅にならないで均一。入力が多くなる

### 注意点
* 開始タグとそれに対応する閉じタグのラインを揃える
