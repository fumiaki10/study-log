## HTMLとは
* Hyper text markup languageの略
* インターネットの大半はHTMLで構成されている

### Hyper Textとは
ウェブページとウェブページをまたぐのに必要な、リンクのことを指す

### markUpとは
```
: <head>や<title>といった要素
: 小文字で書くのが慣習
```

## 基本的な記述
```html
<!DOCTYPE html>　<!--宣言-->
<html>　<!--htmlの範囲-->
	<head> <!--ヘッダー-->
		<meta charset="utf-8">　<!--文字コード-->
		<title>HTMLとは</title>　<!--ブラウザタグについているタイトル-->
	</head><!--ヘッダー閉じタグ-->

	<body>　<!--中身-->
	</body>　<!--中身閉じ-->
</html><!--htmlの終了宣言-->
```
