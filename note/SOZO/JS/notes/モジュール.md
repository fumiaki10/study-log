## 解決方
### importの仕方
```js
// main.js
import { flexBtnLayout } from "./layoutData.js"
```
#### 1.名前付きインポートとデフォルトインポート
```js
// main.js　デフォルトインポート
import flexBtnLayout from "./layoutData"
```
* importの仕方は合わせる必要がある
* 今回は名前付きインポートである上の記述
