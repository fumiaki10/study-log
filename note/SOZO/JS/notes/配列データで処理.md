## 数字ではなく配列データの数によって処理する
```js
 function createButton() {

    let itemsHtml = '';

    flexBtnLayout.forEach(function (value) {
      itemsHtml +=
        `<input type="radio" name="button" data-class=${value.name}>
        <label>${value.label}</label>`;
    });

    flexElement.innerHTML = itemsHtml;
  }
  createButton();
```
```js
  function createButton() {
    let itemsHtml = '';

    for (let i = 1; i <= 5; i++) {
      itemsHtml += `<input type="radio" name="button" data-class=flex-primary${i}>`;
    }

    flexElement.innerHTML = itemsHtml;
  }
  createButton();
```
* 当初flex-primaryという形に数字をくっつける形をとっていた
```js
itemsHtml += `<input type="radio" name="button" data-class=flex-primary${i}>`;
```
これでもまあcssに記述しているやつが、primary-1,2,3と数字が上がって行く形だったので動くのは動くのだが問題があった

**見づらい＋入れ替えられない**

* 純粋にあとで見た時flex-primary１ってなんだ？ってなる
* cssで中身入れ替えて運用は、できるが見栄えも後処理も不便
* ついでに数字を書き換える必要があって、ミスにつながりやすい
### 配列データで扱えるようにした
```js
flexBtnLayout.forEach(function (value) {
      itemsHtml +=
        `<input type="radio" name="button" data-class=${value.name}>
        <label>${value.label}</label>`;
    });

```
* これなら配列データの数だけ繰り返して表示
* cssとjsに必要なデータを加えればいい
* この処理自体に触れることは一切ない

できるだけ見やすく、触れずに、役割を分ける
