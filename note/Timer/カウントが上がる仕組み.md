# 数字が1秒ごとに自動で上がって行く仕組み

## 概要
タイマーアプリを作るときに数字を1秒ごとに動かすようにしたかった
当初ループで処理しようとしたら、基本使わないとのことで
改めて調べた先
・useEffect
・setInterval
この二つを利用すると数字がカウントされていく仕組みが作れることを知った

## useEffect
・副作用（画面の描画とは直接関係ない処理）を実行するための標準フック
・依存配列(下のコードだと[]の部分)が変化したとき実行される
```tsx
 useEffect(() => {
    const intervalId = setInterval(() => {
      setCountTimer((prevCount) => prevCount + 1);
    }, 1000);

    return () => clearInterval(intervalId);
  }, [])
```
## 依存配列
配列無しの時
コンポーネントが再描画される毎回に実行される

空の配列（[]）
コンポーネントが最初に画面に表示された時（マウント時）の一回のみ実行

値を入れた場合(value)
初回時＋指定した値が前回から変化したときのみ実行される。配列の中身が更新されたり、数値が変わったときなど

## setInterval
指定したミリ秒ごとに特定の処理（関数や状態の更新など）を繰り返すJavaScriptの標準関数
よく再レンダリングやメモリリーク（不要なバックグラウンド処理の残留）を防ぐため、useEffectとclearIntervalと組み合わせて記述される
